Полный путь от основ SQL до Production-ready Data Warehouse. От партиционирования данных в PostgreSQL до оптимизации ETL-pipeline на Python и анализа в ClickHouse.

📊 О проекте

StreamFlow — это учебный проект для построения полноценного Data Warehouse аналитики стриминговой платформы. Вы пройдете от глубокого понимания SQL через высокоуровневые паттерны Python ETL до промышленной оркестрации Airflow.

Цель: За 8 недель овладеть полным стеком инструментов современного Data Engineer.

🗺️ Roadmap
  • 📚 Недели 1-2: PostgreSQL + SQL (индексы, оконные функции, MVCC)
  • 🐍 Недели 3-4: Python ETL (генераторы, asyncio, error handling)
  • 📈 Недели 5-6: ClickHouse + OLAP (columnar storage, MergeTree)
  • ⚙️ Недели 7-8: Airflow + Orchestration (DAG, scheduling, production)
