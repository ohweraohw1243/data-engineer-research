Лайв-кодинг кейсы

SQL кейсы для практики. Примеры с подсказками.

📌 Кейс 1: Топ-3 по просмотрам в каждой категории

Задача: Найти топ-3 стримов в каждой категории.
Таблица: streams (id, name, category, views)
Подсказка 1: Используйте RANK() OVER (PARTITION BY category ORDER BY views DESC)
WITH ranked AS ( SELECT *, RANK() OVER (PARTITION BY category ORDER BY views DESC) as rank FROM streams ) SELECT * FROM ranked WHERE rank <= 3;

📌 Кейс 2: Скользящее среднее за 7 дней

Задача: Рассчитать 7-дневное скользящее среднее.
Таблица: daily_stats (date, views)
Подсказка: AVG() OVER (ORDER BY date ROWS BETWEEN 6 PRECEDING AND CURRENT ROW)
SELECT date, views, AVG(views) OVER ( ORDER BY date ROWS BETWEEN 6 PRECEDING AND CURRENT ROW ) as moving_avg_7d FROM daily_stats ORDER BY date;

📌 Кейс 3: Running total (кумулятивная сумма)

Задача: Рассчитать running total доходов.
Таблица: revenue (date, amount)
SELECT date, amount, SUM(amount) OVER (ORDER BY date) as running_total FROM revenue ORDER BY date;
