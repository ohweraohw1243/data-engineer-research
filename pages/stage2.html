Этап 2: Python ETL и Сбор Данных

В этом этапе учимся писать код на Python: делать API-запросы, парсить JSON, подключаться к PostgreSQL и запаковывать Python-скрипты в изолированные Docker контейнеры. Все практические задания перенесены в раздел "Вопросы".

Теоретические модули

🌐 Модуль 1: Python, API и JSON

Чтобы стать Data Инженером, достаточно понимать базовые структуры данных и уметь общаться по протоколу HTTP.

  • Библиотека Requests: Стандарт для HTTP-запросов. Вы делаете requests.get('https://api.example.com/data') и получаете ответ.
  • JSON (JavaScript Object Notation): Главный формат обмена данными. Для Python формат JSON — это просто вложенные Словари (Dicts) и Списки (Lists).

Что посмотреть/почитать:

  • ▶️ YouTube: Лекции про REST API для начинающих, Postman tutorial.
  • 📖 Документация: Официальная документация библиотеки requests в Python (Quickstart).
  • 💻 Практика: Потренироваться дергать открытые API (например reqres.in или pokeapi.co).

🔌 Модуль 2: Подключение Python к Базе Данных

Выступать связующим мостом будет "драйвер базы данных".

Низкоуровневый — psycopg2: Вы открываете соединение и пишете SQL: cursor.execute("INSERT ..."). Важно: для массивов данных используйте executemany().

Высокоуровневый — SQLAlchemy / Pandas: Просто кидаем DataFrame в БД через df.to_sql('table_name').

Что посмотреть/почитать:

  • ▶️ YouTube: Работа с базой данных в Python через psycopg2 (Основы).
  • ▶️ YouTube: SQLAlchemy Core tutorial (базовое подключение и engine).
  • 📝 Статья: Как правильно и быстро делать массовый INSERT (executemany vs execute_values) на Хабре.

🐳 Модуль 3: Docker-изация Python скрипта

Код нужно контейнеризировать, чтобы среда исполнения была изолирована.

Сети в Docker Compose: Внутри контейнера "localhost" — это он сам. Обращайтесь по имени сервиса из docker-compose.yml (например, host="postgres").

Что посмотреть/почитать:

  • ▶️ YouTube: Docker для Python: Как упаковать свой скрипт (Python Dockerfile).
  • ▶️ YouTube: Связывание нескольких контейнеров в Docker Compose (Python + БД).
  • 📖 Практика: Официальный туториал Docker: "Containerize a Python application".
