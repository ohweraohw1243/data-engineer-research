Roadmap — SMT Middle Data Engineer

Детальный план подготовки к вакансии в команду SMT. Стек: Spark, ClickHouse, Kafka, S3, Python. Старт: 3 апреля 2026 года.

План подготовки (Весна - Лето)

📌 Этап 1: Продвинутый SQL & Архитектура БД (Апрель)

Цель: Идеальное владение SQL, оптимизация запросов и принципы DWH/OLAP.

  • DWH, Lake, OLAP: Проектирование схемы (Звезда, Снежинка), Kimball vs Data Vault. Хранение: S3.
  • ClickHouse: Движки семейства MergeTree. Как выжать максимум (Шардирование, партиционирование, Primary Key vs Order By).
  • SQL: Оконные функции, CTE, оптимизация планов запросов (EXPLAIN), индексы.

📌 Этап 2: Промышленный Python & Инженерия (Апрель - Май)

Цель: Писать надежный, конкурентный код под отказоустойчивые пайплайны.

  • Python: Генераторы, асинхронность (asyncio), многопоточность/процессность, ООП в пайплайнах.
  • Отказоустойчивость: Retry/Backoff стратегии, идемпотентность, обработка сбоев.
  • Инфраструктура: Docker, Kubernetes (основы деплоя), CI/CD пайплайны.

📌 Этап 3: Spark & Обработка Big Data (Май)

Цель: Главный инструмент платформы. Уметь "готовить" Spark 3+ на больших объемах.

  • Архитектура: Driver, Executor, DAG, RDD vs DataFrame vs Dataset.
  • Оптимизация: Partitioning, Shuffle, Broadcasting, Skewed data handling, кэширование (persist/cache).
  • Spark 3+ Фичи: Adaptive Query Execution (AQE), Dynamic Partition Pruning. Управление памятью.

📌 Этап 4: Streaming, Kafka & Data Quality (Июнь)

Цель: Высоконагруженная передача данных и контроль их качества.

  • Apache Kafka: Архитектура топиков, партиций, гарантии доставки (At-least-once, Exactly-once), сотни тысяч RPS.
  • Data Governance: Data Catalog, Data Lineage (происхождение данных).
  • Data Quality: Data Contracts, инструменты мониторинга качества данных.

⏳ Этап 5: Алгоритмы, System Design и Проекты (Лето)

Цель: Подготовка к секциям в BigTech, упаковка знаний.

  • System Design: Проектирование Batch/Streaming платформ (от S3 до BI витрин: DataLens, Power BI).
  • Алгоритмы и Математика: LeetCode, оценка сложности, базовый тервер. Вынесены во вкладку "Математика / Лайв-кодинг" для регулярной тренировки.
  • Pet-Project: MVP платформы данных "S3 -> Kafka -> Spark -> ClickHouse" с мониторингом.
