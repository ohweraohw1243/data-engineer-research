Математика и логика

Как математические концепции применяются в SQL и базах данных.

📊 Теория множеств → SQL JOIN

INNER JOIN = Пересечение (A ∩ B)
Только строки в обеих таблицах.

SELECT * FROM A INNER JOIN B ON A.id = B.id;

LEFT JOIN = A + пересечение
Все из A плюс совпадения из B (NULL для несовпадений).

SELECT * FROM A LEFT JOIN B ON A.id = B.id;

FULL OUTER JOIN = Объединение (A ∪ B)
Все из A и все из B (NULL для несовпадений с обеих сторон).

CROSS JOIN = Декартово произведение
Каждая строка A с каждой строкой B. |A| × |B| результатов.

🔍 Логика → WHERE условия

AND (∧): Обе условия истинны.

WHERE age > 18 AND city = 'Moscow';

OR (∨): Хотя бы одна истинна.

WHERE status = 'active' OR status = 'pending';

De Morgan's laws:
NOT (A AND B) = (NOT A) OR (NOT B)

📈 Big O → EXPLAIN cost

O(1): Index Only Scan

O(log n): Index Scan (B-tree)

O(n): Seq Scan

O(n log n): Sort

O(n²): Nested Loop Join
